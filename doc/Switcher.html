Switcher控件

Switcher控件能对鼠标事件进行响应。查看详细

继承自Button。

属性

  • checked : bool
  • background : Loader
  • indicator : Loader

信号

参见Button

方法

参见Button

详细说明

Switcher本身具备按钮的基本特性，默认提供了一个style方案。

这个控件本身不提供复杂的视图，目的在于方便的写出类似Windows的简单复选框。

用法

Switcher {
    id: switcher
}

Button控件能够自动根据内部内容调整大小，因此不必刻意的去设置Button的大小。

属性文档

checked : bool

这个属性用于保存按钮的选中状态。默认值为false。

background : Loader

这个属性控制按钮的状态背景显示。

参见QML Loader Element。

indicator : Loader

这个属性控制按钮的状态显示。

参见QML Loader Element。
