Banner控件

Banner控件提供类似网页横幅功能。查看详细

继承自Item。

属性

  • index : int
  • circleMode : bool
  • timer : Timer

信号

参见QML Item Element。

方法

参见QML Item Element

详细说明

Banner控件提供视图容器，可以控制子视图做横向滑动。

用法

下面这个例子被包含在ExampleBanner.qml:

Banner {
    anchors.centerIn: parent
    Rectangle {
        width: 200; height: 200; color: "lightgray"
        Text { text: "view 1"; anchors.centerIn: parent }
    }
    Rectangle {
        width: 200; height: 200; color: "gray"
        Text { text: "view 2"; anchors.centerIn: parent }
    }
    Rectangle {
        width: 200; height: 200; color: "dimgray"
        Text { text: "view 3"; anchors.centerIn: parent }
    }
}

Banner控件可以自动适应内部视图大小，因此不用显示指定Banner控件的大小，利用此控件你可以很容易做到网页横幅的效果。

属性文档

index : int

这个属性表示当前处于哪个视图，默认为Banner内部子视图的顺序，从0开始计数。

circleMode : bool

这个属性用于控制Banner控件子视图滑动的方式，是圆形轨迹还是顺序轨迹，默认为true，圆形轨迹。

timer : Timer

这个属性用于控制时间间隔以相关特性。

参见QML Timer Element。
