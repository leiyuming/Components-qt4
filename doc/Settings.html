详细说明

Settings对象提供对配置文件的读写，对应方法与QSettings相同。

具体的方法调用，参见QSettings。

唯一不同的是Settings对象在初始化完成后，必须要Component.onCompleted: settings.setFileName()，调用一下setFileName，否则不能正常使用，这个主要是考虑到QSettings可以操作NativeFormat格式的配置。如果setFileName的参数filename为空，默认为NativeFormat操作，否则为IniFormat操作。

用法

Settings { 
    id: settings
    Component.onCompleted: settings.setFileName("config.ini")
}
