  • onIndicatorPosChanged(int dx, int dy)

方法

参见QML MouseArea Element

详细说明

ScrollBar提供滚动条的基本功能，提供滑块操作。

用法

无

属性文档

value : int

滚动条滑块所在位置，取值范围0-100，百分比表示。

step : int

滑块步进长度，像素表示。

background : Loader

这个属性控制滚动条背景。

参见QML Loader Element。

indicator : Loader

这个属性控制滚动条滑块。

参见QML Loader Element。

信号文档

Link::onIndicatorPosChanged(int dx, int dy)

滑块位置变化时触发此信号。

参数dx，dy值大小没啥实际意义，可以通过值的正负来判断方向。
