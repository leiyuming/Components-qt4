IndicatorCircle控件

IndicatorCircle控件提供等待的动画。查看详细

继承自Item。

属性

  • radius : int
  • count : int
  • indicator : Component

信号

参见QML Item Element

方法

参见QML Item Element

详细说明

IndicatorCircle提供一个等待的动画，可以应用于开网页，应用初始化等的费时操作。

网上有很多类似有图所示的loading的GIF图片，但是有时候很想自定义一个，IndicatorCircle则提供了自定义的功能，可以满足小部分需求。

用法

下面这个例子被包含在ExampleIndicatorCircle.qml:

IndicatorCircle {
    indicator: Rectangle { width: 4; height: 4; color: "black"; smooth: true }
}

属性文档

radius : int

这个属性用于控制转动半径。

count : int

这个属性设置指示器小块的数量。

indicator : int

这个属性控制指示器小块的形状，可以是任何可视的QML元素。
