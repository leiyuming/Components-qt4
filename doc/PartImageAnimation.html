PartImageAnimation控件

PartImageAnimation控件能动画显示一个图片组。查看详细

继承自Item。

属性

  • image : Image
  • partCount : int
  • index : int
  • direction : Qt::Orientation
  • timer : Timer

信号

参见QML Item Element

方法

参见QML Item Element

详细说明

PartImageAnimation控件提供了一个帧动画的解决方案，你只需要准备一副帧序列的图片。

用法

下面这个例子被包含在ExamplePartImageAnimation.qml:

PartImageAnimation {
    image.source: "file:ExamplePartImageAnimation/connecting.png"
    partCount: 10
    timer.interval: 100
}

PartImageAnimation是将一个完整的图片做分割显示，对图片源有简单要求

比如：这个图片你需要明确一下几个问题：

  1. 首先这个图片是个帧序列，默认为垂直排列（PartImageAnimation控件的direction默认是垂直的，所以在例子中没有设置， 如果你的图片帧是横排的，那么direction这个值应该被设置为Qt::Horizontal）；
  2. 你需要明确这个图片有多少帧，用partCount写明；
  3. 最后就是动画的时间间隔（你可以不设置，默认为100毫秒）。

你不需要指明PartImageAnimation的大小，当你设置好image.source和partCount后，会自动计算的。

属性文档

image : Image

参见QML Image Element。

partCount : int

这个属性设置图片帧的数量。

index : int

这个属性指示当前帧。

direction : Qt::Orientation

这个属性描述图片源帧的排列方向，默认为Qt.Vertical。

timer : Timer

通过这个属性，我们可以控制帧动画的速率，是否循环播放等状态。

具体使用方法参加QML Timer Element。
