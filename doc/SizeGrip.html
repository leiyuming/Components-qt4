参见Button

详细说明

SizeGrip控件提供改变父对象大小的功能，类似对话框大小调节，默认放在右下角。

用法

下面这个例子被包含在ExampleSizeGrip.qml:

Rectangle {
    width: 100; height: 100
    color: "yellow"
    SizeGrip {}
}

我们可以通过indicator属性对SizeGrip的显示样式进行定制，比如：

indicator： Rectangle {width: 10; height: 10; color: "gray"}

属性文档

minimum : variant

这个属性描述可调节的最小范围。

值需要这样填充：{"width": 100; "height": "100"}。

maximum : variant

这个属性描述可调节的最大范围。

值需要这样填充：{"width": 500; "height": "500"}。

indicator : Component

这个属性控制SizeGrip的显示，可以是任意QML可视元素。
