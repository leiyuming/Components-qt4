LineEdit控件

LineEdit控件提供单行文本输入。查看详细

继承自Item。

属性

  • background : Loader
  • message : Text
  • content : TextInput

信号

  • onTextChanged(string text)
  • onAccepted(string text)

方法

参见QML Item Element

详细说明

LineEdit提供单行文本输入框，默认有一套样式。

我们很多时候需要用到文本输入，但是用QML原生提供的TextInput，我们不得不为其写一些背景和其他控制元素，每次都写的话会觉得繁琐，LineEdit提供了一个默认的解决方案，并且可以灵活定制。

用法

下面这个例子被包含在ExampleLineEdit.qml:

LineEdit {
    width: 300; height: 30
    anchors.centerIn: parent
    message {
        text: qsTr("请在这里输入内容")
        color: "yellow"
    }
    // background.sourceComponent: Rectangle { color: "dimgray" }
}

属性文档

background : Loader

这个属性用于控制输入框的背景。

使用方法参见QML Loader Element。

message : Text

这个属性用于显示输入框的提示信息。

使用方法参见QML Text Element。

content : TextInput

这个属性用于控制输入文本。

使用方法参见QML TextInput Element。

信号文档

LineEdit::onTextChanged (string text)

当输入框文本内容改变的时候会触发此信号。

LineEdit::onAccepted (string text)

在输入框内按下回车键时触发此信号。
