RadioButton控件

RadioButton控件能对鼠标事件进行响应。查看详细

继承自Button。

属性

  • checked : bool
  • title : Text
  • indicator : Loader
  • container : Row

信号

参见Button

方法

参见Button

详细说明

RadioButton本身具备按钮的基本特性，默认提供了一个style方案。

这个控件本身不提供复杂的视图，目的在于方便的写出类似Windows的简单复选框。

用法

下面这个例子被包含在ExampleRadioButton.qml:

RadioButton {
    id: radiobutton
    title {
        text: "RadioButton"
        color: "black"
    }
}

Button控件能够自动根据内部内容调整大小，因此不必刻意的去设置Button的大小。

如果你需要更改indicator的样式，你可以这样做：

RadioButton {
    id: radiobutton
    indicator.sourceComponent: Image { source: radiobutton.checked ? "checked.png" : "unchecked.png" }
}

当然你也可以不用图片，可以用其他元素来作为indicator。

属性文档

checked : bool

这个属性用于保存按钮的选中状态。默认值为false。

title : Text

这个属性控制按钮的文本。

indicator : Loader

这个属性控制按钮的状态显示。

container : Row

这个属性控制按钮内容的布局。

通过这个属性可以方便的调整indicator和title之间的间距。例如:

RadioButton {
    id: radiobutton
    container.spacing: 10
}

这样就间隔了10个像素。
