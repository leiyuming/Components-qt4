Components 参考文档

Components组件包含的控件和方法，已经可以满足绝大部分程序的需求。简单的在QML文件中使用“import Components 1.0”就能做到对基本所有的窗口元素进行定制；除此之外，Components内部提供了对话框，弹出窗口，弹出菜单，以及ToolTip的功能，极大的简化了自定义对话框，窗口的写作过程，并提供了一个简单通信方式，现在基本可以完全通过QML脚本来完成复杂窗口的创建和自定义，非常简单。

在设计Components的时候考虑了比较多的因素，比如控件的自定义，样式控制，与主程序的通信，窗口基本属性控制，多语言问题，并且都提供了相关的解决方法。

写此参考文档的目的主要是为了和同类交流，提高Components的质量，顺便说明各个控件的使用方法。

每个控件的参考文档，我尽量都写一到两个例子来帮助大家理解，在文档中，我将弹出窗口，弹出菜单等内容做一个专题来跟大家交流。（以后有时间再完善，累啊累的）

Components使用说明：传送门

Components组件提供的内容主要包括以下几个方面：

  1. 由原生QML语法写成的基本控件
  2. 由C++提供的扩展控件，主要是解决目前原生QML元素无法提供的功能
  3. 运行时对象（qApp，window，Util，components）

Components基本控件

  • Button - 按钮，提供一些基本功能，比如鼠标指针改变，提示信息
  • CheckBox - 复选框按钮
  • Frame - 图形层（额...不是很有用）
  • FrameAnimation - 帧动画，游戏里用得比较多
  • IndicatorCircle - 圆形指示器
  • IndicatorImage - 图片指示器（额...跟圆形的效果类似）
  • IndicatorLine - 线性指示器（参考Win8做的）
  • Line - 线
  • LineEdit - 输入框
  • Link - 超链接
  • PartImage - 图片局部显示控件（不建议使用，这个主要是提供給PartImageAnimation使用的）
  • PartImageAnimation - 帧动画
  • ProgressBar - 进度条
  • RadioButton - 单选按钮
  • RadioGroup - 单选按钮组
  • Rating - 等级评定控件
  • ScrollArea - 滚动区域
  • ScrollBar - 滚动条
  • ScrollText - 滚动文字
  • Selector - 选择指示器
  • SizeGrip - 控件大小控制器
  • Slider - 滑动器
  • Spliter - 分割线控制器
  • Switcher - 开关按钮
  • Window - 窗口
  • ToolBoxItem - 工具箱项

Components容器控件

  • Banner - 横幅控制容器
  • ChainedView - 链式视图控制器
  • GroupBox - 组
  • StackView - 栈视图控制器
  • TabView - 标签控制器
  • ToolBox - 工具箱

Components扩展控件

  • CursorArea - 鼠标手势控制
  • DropArea - 拖放控制
  • EventLoop - 消息循环控制
  • BlurEffect - 模糊特效
  • ColorizeEffect - 色彩特效
  • DropShadowEffect - 阴影特效
  • OpacityEffect - 透明特效
  • Pixmap - 二进制图片
  • Settings - 配置文件读写
  • WheelArea - 鼠标滚轮控制

Components运行时对象

  • Util - 提供各种实用功能
  • qApp - 全局QApplication对象
  • components - Components全局存储对象（为解决style写的）
  • invoker - 方法调用器，可做全局通信
  • window - 运行时窗口对象
