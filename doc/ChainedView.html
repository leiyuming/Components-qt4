ChainedView控件

ChainedView控件提供一个视图的链式管理。查看详细

继承自Item。

属性

参见QML Item Element。

信号

参见QML Item Element。

方法

  • ChainedView::show(view, parameter)
  • ChainedView::clear()
  • Item ChainedView::getCurrentView()

    • 详细说明

    很多时候，我们会有这样的需求，在一个页面上点击一个按钮后，显示另一个页面，类似向导界面。

    ChainedView控件提供一个类似的功能，不同之处在于，当你切换到“下一页”时，上一个页面会被删除掉，也就是说上一个页面的数据也会消失。

    用法

    下面这个例子被包含在ExampleChainedView.qml:

    ChainedView {
    id: chainedview
    width: 600
    height: 400

    Component {
        id: view1
        Rectangle {
            width: chainedview.width; height: chainedview.height; color: "lightgray"
            Text { text: "view 1"; anchors.centerIn: parent }
            Button {
                anchors { bottom: parent.bottom; right: parent.right }
                Rectangle { width: 40; height: 20; color: "black" }
                onClicked: chainedview.show(view2)
            }
        }
    }

    Component {
        id: view2
        Rectangle {
            width: chainedview.width; height: chainedview.height; color: "gray"
            Text { text: "view 2"; anchors.centerIn: parent }
            Button {
                anchors { bottom: parent.bottom; right: parent.right }
                Rectangle { width: 40; height: 20; color: "black" }
                onClicked: chainedview.show("file:ExampleToolBox.qml", {"width": chainedview.width})
            }
        }
    }
    Component.onCompleted: show(view1)
}

    ChainedView控件支持Component和qml文件。

    方法文档

    ChainedView::show(view, parameter)

    view可以是qml文件地址，也可以是Component对象。

    parameter参数可选，用于view的初始化。

    ChainedView::clear()

    这个方法用于清除当前视图。

    Item ChainedView:getCurrentView()

    返回当前视图对象。
