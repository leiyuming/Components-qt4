Spliter控件

Spliter控件提供改变对象大小的功能 。查看详细

继承自CursorArea。

属性

  • direction : Qt::Orientation
  • range : variant

信号

  • onPositionChanged(int dx, int dy)

方法

参见CursorArea

详细说明

Spliter控件提供改变对象的宽或高。

用法

下面这个例子被包含在ExampleSpliter.qml:

Rectangle {
    width: 100; height: 100
    color: "yellow"
    Spliter {
        id: spliter
        x: parent.width - width
        width: 2; height: parent.height
        onPositionChanged: {parent.width = parent.width + dx}
        Rectangle {
            anchors.fill: parent; color: "black"
        }
    }
}

属性文档

direction : Qt::Orientation

这个属性描述Spliter的方向，默认为Qt.Horizontal。

range : variant

这个属性描述可调节的范围。

值需要这样填充：[0, 1000]， 第一个只为最小值，第二个值为最大值。

信号文档

Spliter::onPositionChanged(int dx, int dy)

当鼠标在Spliter按下并拖动时触发。
