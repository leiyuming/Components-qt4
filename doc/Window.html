Window控件

信号

参见MouseArea

方法

参见MouseArea

详细说明

Window控件能对鼠标事件进行响应，用于弹出对话框，弹出窗口，弹出菜单。

此控件用于QML的最外层，也就是说所有的内容都写在Window{}里面，如果你需要对窗口进行一些特殊控制，Components组件提供了一个运行时对象window，通过这个对象你可以对窗口的基本属性进行设置，比如你想设置窗口的标题栏名称，你可以这样：

window.windowTitle = "window title"

如果你还想设置窗口的其他属性，请参见QWidget属性。

值得一提的是，窗口的大小是根据Window控件的大小来决定的，例如：

Window { width: 800; height: 600 }

这样，实际显示的窗体大小就是800X600了。

用法

无

属性文档

这个属性用于窗口是否可以拖动。
