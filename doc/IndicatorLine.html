IndicatorLine控件

IndicatorLine控件提供等待的动画。查看详细

继承自Item。

属性

  • spacing : int
  • count : int
  • indicator : Component

信号

参见QML Item Element

方法

参见QML Item Element

详细说明

IndicatorLine提供一个等待的动画，可以应用于开网页，应用初始化等的费时操作。

等待动画类似Windows 8 IE 10浏览器加载网页时的效果，其中每一个小点可以通过indicator属性自定义成任何形状。

用法

下面这个例子被包含在ExampleIndicatorLine.qml:

IndicatorLine {
    width: 300; height: 20
    indicator: Rectangle {
        width: 8; height: 8; radius: 4; smooth: true
        color: "dimgray"
    }
}

属性文档

spacing : int

这个属性用于控制线性指示器元素之间的间隔行为。

count : int

这个属性用于控制线性指示器元素的数量。

indicator : Component

这个属性用于定义线性指示器元素的形状。
