Button控件

Button控件能对鼠标事件进行响应。查看详细

继承自MouseArea。

属性

  • __type__ : string
  • enableCursor : bool
  • tip : string
  • cursor : Qt::CursorShape

信号

参见 QML MouseArea Element

方法

参见 QML MouseArea Element

详细说明

Button控件提供的功能与MouseArea一致，并且提供了QML原生的MouseArea没有提供鼠标形状，以及tip信息提示的功能，更适合Windows平台的界面制作。

用法

下面这个例子被包含在ExampleButton.qml:

Button {
    id: button
    Rectangle {
        width: 100; height: 40
        color: button.containsMouse ? "#d2d2d2" : "#b5b5b5"
        Text {
            text: qsTr("按钮")
            anchors.centerIn: parent
        }
    }
}

Button控件能够自动根据内部内容调整大小，因此不必刻意的去设置Button的大小。

属性文档

__type__ : string

这个属性用于描述按钮类型，一般情况下不需要关心。

如果你需要继承Button，那么你应当设置这个属性。

enableCursor : bool

这个属性控制鼠标指针形状是否可以改变。默认值为false。

相关：cursor

tip : string

这个属性控制提示信息。

当鼠标移动到按钮所在区域悬停时，显示此内容提示信息；当信息内容为空时，信息不显示。

cursor : Qt::CursorShape

这个属性控制鼠标指针样式。

注意：如果enableCursor值为false你将看不到鼠标指针改变的效果。
