Slider控件提供一个滑块控制，音量调节，大小缩放常用。

用法

无

属性文档

minimum : int

滑块范围，最小值。

maximum : int

滑块范围，最大值。

如果值为false，将不会做任何处理。

value : int

滑块当前值。

step : int

滑块步进值。

direction : Qt::Orientation

滑块方向，默认为水平方向。

参见QML Loader Element。
