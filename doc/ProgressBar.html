参见QML Item Element

详细说明

ProgressBar控件提供进度显示。简化了QML写进度条的繁琐工作。

此空间提供了背景，块和高亮的自定义，非常简单。

用法

下面这个例子被包含在ExampleProgressBar.qml:

ProgressBar {
    anchors.centerIn: parent
    width: 300; height: 20
    // background: Rectangle { color: "lightgray" }
    // block: Rectangle { color: "lightblue" }
    // highlighter: Rectangle { color: "green" }
}

ProgressBar默认提供了一个style，如果你需要自定义样式，重新写一下background，block，highlighter属性就可以了。

属性文档

background : Component

设置进度条的背景，可以是任何可视元素。

block : Component

设置进度条的背景，可以是任何可视元素。

highlighter : Component

设置进度条的背景，可以是任何可视元素。

highlighterContainer : Item

用于控制当value值为-1的时候处理高亮问题，还没完善，这部分不是很好写。（求指教）

title : Text

用于控制进度条文本显示。

当进度超过title的可视位置的一半时，文本的颜色会根据titleColors属性值进行变更。

请参考QML Text Element。

titleColors : variant

用于控制进度条文本显示的颜色，值的赋值样板为：["color 1", "color 2"]。

value : int

用于控制和记录进度条的进度值。
