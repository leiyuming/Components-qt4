FrameAnimation控件

FrameAnimation控件能动画显示一个图片组。查看详细

继承自Image。

属性

  • url : string
  • fileType : string
  • frameCount : int
  • index : int
  • timer : Timer

信号

参见QML Image Element

方法

参见QML Image Element

详细说明

FrameAnimation控件提供了一个帧动画的解决方案，你只需要将你的图片序列编好号就实现帧动画了。

这样可以避免使用gif图片，如果你要使用gif图片，你不得不带上三方库，这个蛋很疼。

与此控件类似的控件还有：PartImageAnimation这个控件更快更简单，我喜欢。

用法

下面这个例子被包含在ExampleFrameAnimation.qml:

FrameAnimation {
    url: "file:ExampleFrameAnimation/connecting"
    frameCount: 10
    fileType: "png"
    timer.interval: 100
}

我们需要通过url设置图片组的路径以及图片组的名字，路径用“file:路径/图片组名“这样的方法写明。

对于图片组，我们需要手动将图片组里的图片按照”图片组名“ + ”序号“的方式进行命名，序号从”1“开始。

为了让控件能够正常工作，我们需要设置图片组的格式，也就是图片文件的后缀名。

属性文档

url : string

这个属性用于设置图片组的路径以及图片组的名字。

路径用“file:路径/图片组名“这样的方法写明。

fileType : string

这个属性设置图片组的图片格式，与图片组内图片文件后缀名一样。

frameCount : int

这个属性描述图片组的图片个数。

这个值不能大于图片组内图片的个数，可以小于或者是等于图片组内图片的个数。

index : int

这个属性指示当前帧。

timer : Timer

通过这个属性，我们可以控制帧动画的速率，是否循环播放等状态。

具体使用方法参加QML Timer Element。
