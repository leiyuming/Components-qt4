详细说明

Link控件提供类似超链接功能，也可以看成是文本按钮。

用法

下面这个例子被包含在ExampleLink.qml:

Link {
    url: "http://www.baidu.com"
    text: qsTr("百度")
}

属性文档

url : string

这个属性用于保存链接地址。

enableDesktopService : bool

这个属性用于控制是否是使用操作系统默认方式打开链接，默认值为true。

如果值为false，将不会做任何处理。

mouse : MouseArea

这个属性的目的是用于显示的文本样式控制。

比如：color: mouse.containsMouse ? "blue" : "black"

信号文档

Link::onClicked(string url)

当鼠标点击链接时触发此信号。
