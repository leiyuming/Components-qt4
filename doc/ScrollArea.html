ScrollArea控件

ScrollArea控件提供一个滚动区域 。查看详细

继承自Item。

属性

  • controlArea : Flickable
  • centerview : Item
  • horizontalBar : ScrollBar
  • verticalBar : ScrollBar
  • innerScroll : bool
  • horizontalBarPolicy : string
  • verticalBarPolicy : string

信号

参见 QML Item Element

方法

参见 QML Item Element

详细说明

Rating提供一个评分显示和修改的能力。

网上有很多资源都可以对其进行打分，此控件便是模仿哪些打分控件做的，比较容易自定义。

用法

下面这个例子被包含在ExampleScrollArea.qml:

ScrollArea {
    id: scrollarea
    width: 200; height: 200
    innerScroll: false
    Rectangle {
        width: 300; height: 400
        gradient: Gradient {
            GradientStop { position: 0.0; color: "lightsteelblue" }
            GradientStop { position: 1.0; color: "blue" }
        }
    }
}

当然，我们也可以很容易的自定义滚动条的样式，比如我们可以这样做：

verticalBar.background.sourceComponent: Rectangle { color: "black" }
verticalBar.indicator.sourceComponent: Rectangle { color: "blue" }

ScrollBar的使用方法，参见ScrollBar控件。

属性文档

controlArea : Flickable

参见QML Filckable Element。

centerview : Item

ScrollAreak控件会根据centerview的大小自动控制滚动条的动作。

horizontalBar : ScrollBar

这个属性控制水平滚动条的动作。

参见ScrollBar。

verticalBar : ScrollBar

这个属性控制垂直滚动条的动作。

参见ScrollBar。

innerScroll : bool

这个属性控制滚动条是在滚动区域内部，还是在滚动区域外部。默认值为true。

true表示滚动条在滚动区域内部，如果滚动条在滚动区域内部，则默认不显示滚动条，只有在内容滚动的时候才显示。

horizontalBarPolicy : string

这个属性控制水平滚动条的显示是自动，总是显示，还是不显示。

可能取值为"auto", "always", "never"，默认值为"auto"。

注意：这个属性在innerScroll值为false时有效。

verticalBarPolicy : string

这个属性控制水平滚动条的显示是自动，总是显示，还是不显示。

可能取值为"auto", "always", "never"，默认值为"auto"。

注意：这个属性在innerScroll值为false时有效。
