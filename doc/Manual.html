Components 使用文档
  Version: 1.0

Components 是为使用Qt/QML图形库进行开发软件的开发人员设计的QML基础元件库, 为开发人员节省大量时间。 下面我们将要介绍它的一些基本环境要求、如何在开发过程中使用Components库、 以及如何发布带有Components元件的软件产品。

Components的基本要求

  • Qt 4.8.4
  • QtQuick 1.1

向Qt中导入Components库的步骤：

  • step1: 解压。 解压 xxx.zip
  • step2: 移动文件夹。 把刚才解压的文件夹下的 ./Components 文件夹拷贝到 Qt 根目录的 imports 目录下。
  • step3: 开发。 做完上述2个步骤即可以开始使用Components进行开发了。

使用Components库开发的步骤：

  • step1: 引入Components库。 
    import QtQuick 1.1
import Components 1.0

Rectangle {
...
...

发布带有Components控件的可执行文件步骤

  • step1: 生成。 生成exe可执行文件。
  • step2: 拷贝。 拷贝Qt根目录下的 imports 文件夹下的./Components 文件夹(即Components库)到 exe文件目录下。
