Line控件

Line控件提供类似分割线的功能。查看详细

继承自Item。

属性

  • direction : Qt::Orientation
  • content : Component

信号

参见QML Item Element

方法

参见QML Item Element

详细说明

在设计界面的时候，很多情况需要用线来分隔区域，Line控件提供了这个功能。

之所以提供这个控件，是因为有时候我们用QML Rectangle来划线的时候，遇到渐变线并且想要横着放的时候，如果将此Rectangle放到布局控件里（比如：Column），在布局上将很难处理，Line控件则解决了这样的问题，可以很方便的被布局。

用法

下面这个例子被包含在ExampleLine.qml:

Line {
    width: 300; height: 2
    content: Rectangle { color: "black" }
}

属性文档

direction : Qt::Orientation

这个属性用于控制线的方向，默认为Qt::Horizontal。

content : Component

这个属性用于控制线的内容，可以是任意可视元素。
