IndicatorImage控件

IndicatorImage控件提供等待的动画。查看详细

继承自Image。

属性

  • animation : NumberAnimation

信号

参见QML Image Element

方法

参见QML Image Element

详细说明

IndicatorImage提供一个等待的动画，可以应用于开网页，应用初始化等的费时操作。

只需要一个环状图片，就可以做到类似有图所示效果

用法

下面这个例子被包含在ExampleIndicatorImage.qml:

IndicatorImage {
    source: "../resource/loading.png"
}

属性文档

animation : NumberAnimation

这个属性用于控制指示器行为。

使用方法参见QML NumberAnimation Element。
