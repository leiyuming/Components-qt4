方法

参见 QML Item Element

详细说明

Rating提供一个评分显示和修改的能力。

网上有很多资源都可以对其进行打分，此控件便是模仿哪些打分控件做的，比较容易自定义。

用法

下面这个例子被包含在ExampleRating.qml:

Rating {
    id: rating
    enableEdit: true
}

我们可以很方便的通过background和foreground来更改Rating内部元素的样式，他们可以是任意可视QML元素。

属性文档

total : int

这个属性描述可以评定范围，默认值为5。

value : int

这个属性描述当前评分。

enableEdit : bool

这个属性控制是否可以进行评定动作，默认为false。

spacing : int

这个属性控制Rating内部元素间距。

background : Component

这个属性控制Rating内部元素背景，可以是任意QML可视元素。

foreground : Component

这个属性控制Rating内部元素前景，可以是任意QML可视元素。
